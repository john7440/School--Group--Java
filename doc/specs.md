### Pour le directeur, l'application permet :

#### Gestion des élèves

- ajouter un élève à la liste des élèves 
- afficher la liste des élèves 
- consulter le détail d’un élève :
    - voir toutes les informations de l’élève : identité, âge, adresse 
- modifier un élève :
    - mettre à jour les informations d’un élève : nom, prénom, âge, adresse, etc. 
- supprimer un élève :
    - supprimer un élève de la liste 
- valider l'inscription d'un élève :
    - transformer un candidat en élève inscrit.

#### Gestion des enseignants

- afficher la liste des enseignants :
    - afficher l’ensemble des enseignants : prénom, nom, âge, date d’arrivée, adresse 
- consulter le détail d’un enseignant :
    - voir toutes ses informations : identité, adresse, date d’arrivée dans l’école, liste de ses cours
- ajouter un enseignant :
    - saisir : prénom, nom, âge, adresse, date d’arrivée dans l’école 
- modifier un enseignant :
    - mettre à jour les informations d’un enseignant, y compris l’adresse 
- supprimer un enseignant :
    - supprimer un enseignant 

#### Gestion des cours

- afficher la liste des cours :
    - afficher : dénomination du cours, date de début, date de fin, enseignant responsable 
- consulter le détail d’un cours :
    - voir toutes les informations du cours : dénomination, dates, enseignant, liste des élèves inscrits
- ajouter un cours :
    - créer un nouveau cours avec :
        - dénomination du cours 
        - date de début 
        - date de fin 
        - enseignant assigné 
- modifier un cours :
    - mettre à jour la dénomination, les dates, l’enseignant assigné 
- supprimer un cours :
    - supprimer un cours 
- afficher les adresses.

### Pour l'élève, l'application permet :

- faire une demande d'inscription :
    - saisir ses informations personnelles : prénom, nom, âge, adresse 
    - la demande est enregistrée dans le système comme « demande en attente » 
    - le directeur devra ensuite la valider ou éventuellement la refuser
- consulter la liste des cours :
    - voir tous les cours proposés dans l’école : dénomination, dates, enseignant responsable.

### Pour l'enseignant, l'application permet :

- afficher les élèves d'un cours :
    - afficher, pour l’enseignant :
        - dénomination des cours 
        - dates de début et de fin 
- consulter la liste des cours auxquels il est assigné :
    - afficher, pour l’enseignant :
        - dénomination des cours 
        - dates de début et de fin.
